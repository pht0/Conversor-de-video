# Conversor de Vídeos do YouTube

Este projeto é um aplicativo GUI simples desenvolvido em Python, que permite ao usuário baixar vídeos e áudios do YouTube. Utiliza a biblioteca `tkinter` para a interface gráfica e a biblioteca `pytubefix` para o download dos vídeos.

## Funcionalidades

- **Baixar Vídeos**: Insira a URL do vídeo do YouTube e baixe-o no formato MP4.
- **Baixar Áudios**: Insira a URL do vídeo do YouTube e baixe-o no formato MP3.
- **Tema Escuro**: Alterna entre um tema claro e um tema escuro para a interface.
